---

This guide walks you through cloning the repository, installing dependencies, configuring environment variables, and starting both servers.

<Note>
  S-mart requires **Node.js v20.19.0 or later** (v22.12.0+ recommended). Check your version with `node --version` before proceeding.
</Note>

<Warning>
  Both the `Frontend` and `backend` directories require a `.env` file with valid Supabase credentials before the application will work. Complete [Environment Setup](/environment-setup) before running the dev servers.
</Warning>

## Run S-mart locally

<Steps>
  <Step title="Clone the repository">
    Clone the S-mart repository from GitHub and move into the project directory.

    ```bash
    git clone https://github.com/Hiramrr/S-mart.git
    cd S-mart
    ```
  </Step>

  <Step title="Install frontend dependencies">
    Navigate to the `Frontend` directory and install the Node.js dependencies.

    <CodeGroup>

    ```bash npm
    cd Frontend
    npm install
    ```

    ```bash yarn
    cd Frontend
    yarn install
    ```

    </CodeGroup>
  </Step>

  <Step title="Install backend dependencies">
    Open a second terminal, navigate to the `backend` directory, and install its dependencies.

    <CodeGroup>

    ```bash npm
    cd backend
    npm install
    ```

    ```bash yarn
    cd backend
    yarn install
    ```

    </CodeGroup>
  </Step>

  <Step title="Configure environment variables">
    Create `.env` files in both `Frontend/` and `backend/` with your Supabase credentials. See [Environment Setup](/environment-setup) for the exact variables and where to find them in the Supabase dashboard.
  </Step>

  <Step title="Start the backend server">
    From the `backend` directory, start the Express API in development mode.

    ```bash
    npm run dev
    ```

    The backend will be available at **http://localhost:3000**.
  </Step>

  <Step title="Start the frontend dev server">
    From the `Frontend` directory, start the Vite development server.

    ```bash
    npm run dev
    ```

    The frontend will be available at **http://localhost:5173** with hot module replacement enabled.
  </Step>
</Steps>

## Verify the setup

Once both servers are running:

1. Open **http://localhost:5173** in your browser.
2. You should see the S-mart login page.
3. Sign up or log in with a Supabase-authenticated account.
4. Your role (Customer, Seller, Cashier, or Administrator) determines which views are available after login.

## Available scripts

### Frontend (`Frontend/`)

| Script | Description |
| --- | --- |
| `npm run dev` | Start the Vite development server with HMR |
| `npm run build` | Compile the app for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint and auto-fix issues |
| `npm run format` | Format source files with Prettier |

### Backend (`backend/`)

| Script | Description |
| --- | --- |
| `npm run dev` | Start the Express server with nodemon (auto-restart on change) |
| `npm start` | Start the Express server in production mode |
